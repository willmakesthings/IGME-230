Command Description
pwd print working directory
man display system documentation
ls list directory contents
ls -l use a long listing format
ls -la use long listing format and display hidden entries
cp file1 file2 copies file1 to file2
mv file1 file2 move (rename) file1 to file2
cd change directory
cd folder1 changes current directory to folder1
cd .. changes to parent directory
cd / changes to root directory
chmod 755 folder1 changes folder1 permissions to

755 rwxr-xr-x
chmod 644 file1 changes file1 permissions to

644 rw-r--r--
chmod 700 folder1 changes folder1 permissions to

700 rwx------
mkdir folder1 makes new directory folder1
rmdir folder1 removes empty directory folder1
rm file1 removes file1 from current directory
rm proj?.html removes proj?.html from current directory
rm * removes all files from current directory
cal prints a calendar
cal prints a calendar
cal 5 2016 prints a calendar of May 2016
cat file1 print file1
cat file1 file2 >> file 3 concatenates file1 and file2 into file3
nano file1.txt opens file1 in the GNU nano text editor
echo "goop" >> file3 prints "goop" into file3
telnet towel.blinkenlights.nl plays ASCII Star Wars: A New Hope
w/who display who is on the system
finger userName find more information about userName
wc -l file prints file line count
wc -c file prints file byte count
wc -w file prints file word count
write userName send message to userName
